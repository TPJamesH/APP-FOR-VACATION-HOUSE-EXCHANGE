The error message "cannot overload functions distinguished by return type alone" 
means that somewhere in your code, 
there are two or more functions 
with the same name and parameter list
 but different return types. 
 This is not allowed in C++ 
 because the compiler cannot distinguish between them
  based solely on their return types.

  WRONG
  ---
  int getValue();
double getValue(); // Error: cannot overload functions distinguished by return type alone

-----------/-/----------------
PASS BY VALUE 
void processAccount(Account account) {
    // Modify or use the copied Account object
    std::cout << "Processing account: " << account.getFullName() << std::endl;
}

PASS BY REFERENCE
void processAccount(Account& account) {
    // Modify or use the original Account object
    account.setFullName("Updated Name");
    std::cout << "Processing account: " << account.getFullName() << std::endl;
}

PASS BY CONSTANT REFERENCE
void processAccount(const Account& account) {
    // Use the Account object but do not modify it
    std::cout << "Processing account: " << account.getFullName() << std::endl;
}
